# Sales Analysis Dashboard (Excel)

## Project Overview
This project analyzes retail sales data using advanced Excel features to identify top customers, product performance, and profitability across different categories and regions. An interactive dashboard was created to explore sales insights using slicers and pivot charts.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- SUMIFS, COUNTIFS, IF formulas

## Key Analysis
The dashboard analyzes:

- Top 10 customers by total sales
- Sales performance by product sub-category
- Profitability by product sub-category
- Sales segmentation by region, category, and customer segment

## Key Insights
- Certain sub-categories generate high sales but lower profit margins.
- Some products such as **Tables** show negative profitability.
- Sales performance varies significantly across customer segments and regions.

## Dashboard Features
The Excel dashboard includes:

- **Top 10 Customers by Sales**
- **Sales by Sub-Category**
- **Profit by Sub-Category**
- **Interactive Filters (Slicers)** for:
  - Region
  - Category
  - Segment
